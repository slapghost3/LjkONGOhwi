# 项目简介

## 核心功能
- 数据清洗：包括缺失值处理、异常值处理
- 数据预处理：特征选择与提取、类别处理
- 数据统计
- 数据可视化
- 研究结论

## 适用场景
- 市场需求分析
- 行业趋势研究
- 求职者技能提升指导
- 城市就业选择参考

## 技术特点
- 使用 Python 进行数据分析与可视化
- 采用 pyecharts、pandas、numpy 等工具
- 提供数据集、源码、PPT 和论文
- 包含 2024 年全国岗位招聘信息数据分析结果
